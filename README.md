# Система инвентаризации для завода

Приложение для учёта товаров и проведения инвентаризации на производственном предприятии. Разработано на Python с использованием Tkinter.

## 🚀 Функционал

- ✅ Авторизация с разграничением прав доступа (4 роли)
- ✅ CRUD операции с товарами (добавление, редактирование, удаление)
- ✅ Проведение инвентаризаций с фиксацией расхождений
- ✅ Визуализация данных (графики Matplotlib)
- ✅ Генерация QR-кодов для товаров
- ✅ Экспорт данных в CSV/Excel
- ✅ Аудит-лог всех действий пользователей
- ✅ Поиск и фильтрация товаров
- ✅ Настраиваемый конфигурационный файл

## 👥 Роли пользователей

| Роль | Логин | Пароль | Описание |
|------|-------|--------|----------|
| Администратор | `admin` | `admin123` | Полный доступ ко всем функциям |
| Менеджер склада | `manager` | `manager123` | Управление товарами и инвентаризациями |
| Кладовщик | `user` | `user123` | Проведение инвентаризаций и просмотр |
| Аудитор | `auditor` | `auditor123` | Просмотр отчётов и аудит-лога |

## 🔧 Установка и запуск

### Быстрый старт (3 шага)

#### 1️⃣ Скачайте проект


git clone https://github.com/ВАШ_ЛОГИН/inventory-system.git
cd inventory-system
Или просто скачайте ZIP-архив с GitHub и распакуйте.

2️⃣ Установите зависимости

pip install -r requirements.txt
Если нет pip, установите вручную:


pip install matplotlib qrcode Pillow openpyxl

3️⃣ Запустите приложение

python main.py

Готово! Откроется окно авторизации.
